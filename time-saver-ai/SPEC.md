## What TimeSaverAI Does
TimeSaverAI is an AI app that learns your repetitive workflows by watching you do them once, then runs them autonomously on schedule so you never have to do the same task twice.

## Screens

### Screen 1 — Dashboard (Home)
- Hero: "You saved 47 hours this month" with animated counter
- "Your AI learned 12 workflows this week"
- Recent activity feed: workflows completed with timestamps
- Quick action: "Teach AI something new" button
- Bottom nav: Dashboard | Workflows | Activity | Settings

### Screen 2 — Teach Mode
- Full-screen walkthrough recorder
- Shows "AI is watching..." while user does task
- On completion: "I learned that!" with step-by-step replay
- User can approve/edit/delete any step
- "Run automatically" toggle

### Screen 3 — My Workflows
- Grid of workflow cards
- Each card: name, icon, last run, status badge (active/paused)
- Tap to see full step-by-step breakdown
- Toggle: manual run | scheduled run
- "+" button to teach new workflow

### Screen 4 — Activity Feed
- Timeline of all automated actions
- Each entry: workflow name, what it did, timestamp
- "View replay" button to watch AI complete the task
- Filter: today | this week | all time

### Screen 5 — Settings
- Account (email, plan)
- Notifications (telegram, email)
- Connected apps (Gmail, Google Sheets, Calendar)
- Time Saved stats export
- Help | Privacy | Terms

## Tech Stack
- Zo Space (React + Tailwind)
- Hermes Agent (workflow execution)
- Paperclip (skill library)
- Ollama (local reasoning)
