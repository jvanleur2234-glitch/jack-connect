# Transaction Tracker Skill

Track every moving part of a real estate transaction.

## What to Track

For each transaction, track:
- **Property address**
- **Client name**
- **Closing date**
- **All contingencies** (inspection, appraisal, financing, sale of existing home)
- **Deadlines** for each contingency
- **Signatures needed** and status
- **Funds** (earnest money, closing costs)
- **Notes** from all conversations

## Daily Check

Every morning, check all active transactions and alert Jack to:
1. Any deadlines within 48 hours
2. Contingencies that need follow-up
3. Missing signatures
4. Any deals that need immediate attention

## CRM Integration

Keep the JackConnect CRM updated with every status change.
