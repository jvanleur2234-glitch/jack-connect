## PART 5: The Install Process (1 min)

### One Command (Server)
```bash
curl -sSL https://solomon.os/install | bash
```
Steps:
1. Detects hardware (CPU, RAM, GPU)
2. Installs Ollama + models
3. Installs BitNet
4. Installs TileLang
5. Installs FreeLLMAPI
6. Sets up Telegram bot
7. Starts all agents

### One Click (Client)
```bash
# JackConnect.exe — double click
```
- Auto-detects Solomon OS server
- Connects Telegram
- Shows all running agents
- System tray icon appears

## PART 6: How Watch Once Actually Works (2 min)

### Example: New Zillow Lead Email

**Step 1: Record**
Jack clicks "Start Recording" in Watch Once
Jack does the task manually:
1. Opens Gmail
2. Clicks compose
3. Types lead name
4. Pastes property address
5. Writes personalized email
6. Sends

**Step 2: Review**
JackConnect shows:
```
Step 1: Open Gmail → ✓ Approved
Step 2: Click compose → ✓ Approved
Step 3: Type lead name → ✓ Approved
Step 4: Paste property address → ✓ Approved
Step 5: Write personalized email (AI-generated) → ✓ Approved
Step 6: Send email → ✓ Approved
```

**Step 3: Schedule**
Jack sets: "Run automatically when new Zillow lead comes in"

**Step 4: Done**
Every new Zillow lead now gets a personalized email — automatically — forever.
