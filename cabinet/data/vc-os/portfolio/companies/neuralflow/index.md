## Competitive Intel

See [[competitive-intel]] for detailed landscape.

Key pressure: OpenAI's Assistants API + function calling now covers 60–70% of horizontal enterprise integration use cases NeuralFlow targets. The horizontal layer is commoditizing. Moat must come from:
1. **Regulated vertical depth** — legal, healthcare, finance workflows where OpenAI won't operate
2. **Enterprise data connectors** — proprietary integrations with Salesforce, SAP, Oracle that LLM APIs don't touch
3. **Governance & audit trail** — HIPAA/SOC2/ISO27001 compliance layer that enterprises require
