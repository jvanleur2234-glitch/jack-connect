## Business Model

1. **Hardware lease:** IoT sensor arrays deployed at industrial site (~$80K setup + $24K/year maintenance)
2. **Credit origination fee:** 8% of carbon credit value on first issuance
3. **SaaS dashboard:** $12K/year for corporate sustainability reporting platform
4. **Enterprise API:** $36K/year for Scope 3 reporting data feeds
