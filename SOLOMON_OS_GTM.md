## WATCH ONCE ENGINE — THE CORE DIFFERENTIATOR

### How it works:
1. User says "I need to create a CMA report"
2. Watch Once records screen + action (what they click, type, navigate)
3. AI analyzes the workflow: "They opened MLS → searched address → pulled comps → wrote in Word → emailed to client"
4. Skill generated: "create_cma_report.py"
5. Superintendent learns it → next time they ask, it's done automatically
6. Bonus: Skill goes to marketplace for other users

### Why this is the moat:
- Single interaction → permanent automation
- Every new user = more skills in marketplace
- Competitors can't copy "watch once" because they don't have the infrastructure
- Network effect: more users = more skills = more users
